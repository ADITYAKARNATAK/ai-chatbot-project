# ai-chatbot-project
# 🤖 AI Chatbot using NLP & Machine Learning

## 📌 Project Overview

This project is an intelligent AI Chatbot built using **Python, Natural Language Processing (NLP), and Machine Learning**.
It can understand user queries, classify their intent, and respond appropriately through a web interface.

The chatbot is deployed online and accessible through a browser.

---

## 🌐 Live Demo

👉 https://ai-chatbot-project-ofiu.onrender.com

---

## ⚙️ How It Works

1. User enters a message
2. Text is processed using NLP techniques (tokenization, stemming)
3. Converted into Bag of Words
4. Machine Learning model predicts intent
5. Response is selected from predefined intents
6. Response is displayed on the web interface

## 🛠️ Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/ADITYAKARNATAK/ai-chatbot-project.git
cd ai-chatbot-project
```

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   (Windows)
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Train Model

```
python train.py
```

### 5. Run Application

```
python app.py
```

### 6. Open in Browser

```
http://127.0.0.1:5000
```
